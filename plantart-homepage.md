# 5. APPROACH

## 5.1 Eyebrow de sección
```
Metodología
```
o:
```
How we work
```

## 5.2 Título de sección
```
De la estrategia a la operación permanente.
```
o:
```
Un proceso diseñado para entornos exigentes.
```

## 5.3 Subtexto de sección
```
Cada proyecto atraviesa un proceso riguroso que integra diseño, ejecución
y operación continua. No entregamos un jardín; diseñamos un sistema vivo
que funciona en el tiempo.
```

---

## 5.4 Pasos de la metodología

### Paso 1 — Strategy
```
Eyebrow: 01
Título: Diagnóstico y estrategia
Body:
Evaluamos el activo, los requerimientos del cliente y las condiciones del espacio
para definir una solución coherente con arquitectura, marca y uso real.
```

### Paso 2 — Design
```
Eyebrow: 02
Título: Diseño integrado
Body:
Desarrollamos propuesta de diseño biofílico adaptada al espacio, al programa
y al estándar visual del cliente. Selección de especies, volumetría y sistema técnico.
```

### Paso 3 — Build
```
Eyebrow: 03
Título: Ejecución especializada
Body:
Implementamos con equipo propio y supervisión técnica permanente.
Cada instalación se realiza con estándares de calidad verificables
y cronograma definido contractualmente.
```

### Paso 4 — Operate
```
Eyebrow: 04
Título: Puesta en marcha
Body:
Activamos sistemas de riego, verificamos comportamiento vegetal
y ajustamos todos los parámetros técnicos antes de entregar el espacio.
```

### Paso 5 — Maintain
```
Eyebrow: 05
Título: Mantenimiento continuo
Body:
Operación continua con cronograma, supervisión especializada
y criterios de servicio trazables. La naturaleza que instalamos
es también la naturaleza que cuidamos.
```
